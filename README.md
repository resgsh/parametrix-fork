# Parametrix – Charli3-Powered Parametric Risk Pools on Cardano

Parametrix is a decentralized protocol for creating and participating in **parametric risk pools** on Cardano. Users define real-world events (e.g. rainfall thresholds, flight delays), fund premium pools, and allow others to **subscribe to the pool** by underwriting that risk.

At settlement, outcomes are resolved deterministically using Charli3 pull oracles — enabling **trustless, instant payouts without claims processing**.

---

## Core Features

* Create event-linked risk pools
* Subscribe to pools (earn premium yield)
* Oracle-driven settlement (no intermediaries)
* Fully on-chain, transparent logic

---

## How It Works

* **Create Pool**: Hedger defines event + deposits premium
* **Subscribe**: Participants underwrite risk by subscribing capital
* **Settle**: Oracle resolves outcome → automatic payout

---

## Use Cases

* Weather hedging (rainfall, temperature)
* Travel disruption protection
* Event-based financial contracts
